# 📊 Diagnóstico de Rendimiento Comercial – eCommerce Regional
### Proyecto Integrador – Módulo 9: Fundamentos de la Narrativa de Datos

## 📝 Descripción
Este proyecto corresponde al **Módulo 9** del diplomado y se enfoca en la **narrativa de datos** aplicada a un caso organizacional.  

A partir de un caso de eCommerce regional, se construyó una presentación ejecutiva orientada a comunicar un **insight clave de negocio**, utilizando una estructura narrativa clara, apoyo visual y recomendaciones accionables.

## 🎯 Objetivo
Identificar un hallazgo relevante dentro de los datos comerciales y comunicarlo de manera comprensible para facilitar la toma de decisiones.

En este caso, el análisis detecta una **alta concentración de ventas en Santiago de Chile**, lo que representa una dependencia comercial significativa respecto de una sola ciudad.

## 🧠 Estructura narrativa utilizada
La presentación fue desarrollada siguiendo una secuencia lógica de storytelling:

1. **Contexto**  
   Se presenta el escenario general del eCommerce y su operación regional.

2. **Problema**  
   Se identifica la posible dependencia excesiva de una sola ciudad como riesgo comercial.

3. **Insight**  
   Se muestra visualmente que Santiago de Chile concentra la mayor parte de las ventas.

4. **Recomendación**  
   Se proponen acciones para diversificar el mercado y reducir el riesgo estratégico.

## 📊 Insight principal
El análisis evidencia que **Santiago de Chile concentra la mayor proporción de ventas del negocio**, mientras que el resto de las ciudades presenta una participación considerablemente menor.

Este hallazgo sugiere una estructura comercial desequilibrada, con riesgo de dependencia geográfica.

## ✅ Conclusión
La narrativa de datos permite transformar un conjunto de cifras en una historia clara, útil y orientada a la acción.  
En este proyecto, la visualización y la estructura narrativa ayudan a comunicar de forma efectiva un problema de concentración comercial y a proponer medidas concretas para enfrentarlo.
